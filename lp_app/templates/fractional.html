About Fractional Programming

Fractional programming deals with optimizing ratios of functions, such as:

max/min f(x) = g(x)/h(x)

This is useful for problems involving efficiency or rates, such as:

  • Return on investment (profit/cost)
  • Efficiency ratios (output/input)
  • Average cost per unit (total cost/units)
  • Time efficiency (output/time)

This solver uses the Charnes-Cooper transformation method to convert the fractional problem into a standard linear program that can be solved using the simplex method.
